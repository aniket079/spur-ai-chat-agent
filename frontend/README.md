# SwiftCart AI Chat Frontend

A production-quality React chat interface for the SwiftCart AI support agent.

## Features

- **Full-page layout** with left sidebar and main chat area
- **Conversation history** with localStorage persistence
- **Responsive design** that works on desktop and mobile
- **Real-time messaging** with loading states
- **Auto-scroll** to latest messages
- **Error handling** with user-friendly notifications
- **FAQ suggestions** on empty state
- **Clean, modern UI** with pure CSS styling

## Setup

```bash
cd frontend
npm install
npm run dev
```

The dev server will proxy API calls to `http://localhost:4000`.

## Build

```bash
npm run build
```

Output is in the `dist/` folder.
